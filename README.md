# How to run it

## 1. unzip the `DEMO_PROJECT_Fixed.zip` file.

## 2. Enter the folder `DEMO_PROJECT_Fixed`:-

      ```
      cd DEMO_PROJECT_Fixed/
      ```

## 3. Run the application using docker-compose:

      ```
      docker-compose up -d --build
      ```

## 4. Access the Application:-

      ```
      http://localhost:5000/
      ```
